组件定义
默认情况下，app 内的组件是 React 服务器组件，如果要使用客户端组件，在页面顶部用'use client'指令
一旦定义了客户端组件，其子组件都被视为客户端组件
服务端组件里可以包含客户端组件，但是客户端组件里不能直接导入服务端组件，可以通过children prop的方式讲服务端组件在客户端组件中使用。

数据请求
对于服务端组件，可以使用fetch api封装请求，Next.js 扩展了原生的 fetch Web API，允许你为服务器上的每个获取请求配置 缓存 和 重新验证 行为。你可以在 服务器组件、路由处理程序 和 服务器操作 中使用 fetch。
对于客户端组件，使用数据获取库，如 SWR 或 React Query 来获取数据，这些库提供了自己的 API 用于缓存、重新验证和变更数据。
